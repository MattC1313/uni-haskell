### Homework topic summaries
Here is a brief of what topics each homework covers:
- *week1.hs*
  - what is a pure function
  - loading in files to the ghci
  - making my own functions
- *week2.hs*
  - The 'if' and 'let' statements
  - Tuples and lists
  - List functions
  - List ranges and comprehension (includes FizzBuzz)
- *week3.hs*
  - Recursion
  - List recursion
- *week5.hs*
  - Partial application
  - Polymorphic types
  - Anonymous functions
  - Function composition (`.` and `$`)
  - `map` and `filter`
- *week6.hs*
  - `fold`
  - `scan`
  - `takeWhile` and `dropWhile`
  - `zipWith`
- *week7.hs*
  - Custom types
  - `Maybe`
  - `Either`
- *week8.hs*
  - Introduction to IO
  - `getLine` and `putStrLn`
  - `Do` blocks
  - Looping in IO code
  - Compiling and running programs
- *week10.hs*
  - Lazy evaluation
  - Lazy evaluation on lists
  - Tail recursion
